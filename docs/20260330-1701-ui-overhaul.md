## 2. デザイン方針

参考画像から読み取った特徴:
- **チケット風カードデザイン**（切り離し風の点線、搭乗券レイアウト）
- **ボトムナビ**（Booking / Tickets 等のアイコン付き）
- **クリーンな白ベース + アクセントカラー**
- **ラベル+値の対で情報を整理**
- **角丸カード + 軽い影**
- **大きなCTAボタン**

適用した方針:
- メインカラー: 淡いオレンジ `#e8956a`
- 背景: `#faf8f6`（暖かみのあるオフホワイト）
- ヘッダー: absolute top、ロゴ中央のみ（最小限）
- フッター: 固定ボトムナビ（Tabler icons）
- 残席数は非表示（「予約受付中」「残りわずか」「完売」の3段階のみ）
- Tabler Icons を全面導入
- チケット風カード（点線切り離し）を確認画面・予約確認画面に

## 4. デザインの具体的変更

### 4.1 カラーシステム

| 変数 | 値 | 用途 |
|------|-----|------|
| --mogi-orange | #e8956a | メインアクセント |
| --mogi-orange-light | #fdf0ea | 選択状態の背景 |
| --mogi-orange-dark | #c4714a | ボタンホバー |
| --mogi-text | #2d2a26 | テキスト |
| --mogi-text-muted | #8a8580 | 補助テキスト |
| --mogi-bg | #faf8f6 | ページ背景 |
| --mogi-border | #ece8e4 | カード枠線 |

### 4.2 ヘッダー

- `position: absolute; top: 16px` でコンテンツに重ならずフロート
- ロゴ（logo.svg）中央配置、高さ 24px、opacity 0.7

### 4.3 フッター（ボトムナビ）

- `position: fixed; bottom: 0` で常時表示
- 3項目: ホーム（IconHome）、チケット（IconTicket）、受付（IconUsers）
- アクティブ時にオレンジ色
- safe-area-inset-bottom 対応（iPhone ノッチ）
- #app に padding-bottom: 72px で本文がフッターに隠れない

### 4.4 残席表示の変更

**Before:** `残 7/10` のように具体数を表示
**After:** 3段階のみ
- 「予約受付中」（20%超）→ オレンジ
- 「残りわずか」（20%以下）→ アンバー
- 「完売」（0）→ グレー

### 4.5 チケット風カード

ReservationSummary と ReservationConfirmView で採用:
- 上部: 予約情報（公演、席種、枚数、支払方法、合計金額）
- 点線切り離し（dashed border + 左右に丸い切り欠き）
- 下部: お客様情報

### 4.6 席種選択

**Before:** list-group のリスト
**After:** カード型ボタン。選択中はオレンジ枠 + オレンジ淡背景。アイコン付き。

### 4.7 枚数選択

**Before:** select ドロップダウン
**After:** 横並びボタン（1, 2, 3...）。選択中はオレンジ塗り。
